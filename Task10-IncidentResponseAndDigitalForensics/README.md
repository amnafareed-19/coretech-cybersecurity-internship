# Task 10 - Incident Response and Digital Forensics

## Objective

The objective of this task was to learn the fundamentals of incident response and digital forensics. The task included understanding the incident response lifecycle, developing a ransomware incident response plan, learning evidence handling procedures, performing basic forensic analysis using Autopsy, and studying log analysis and SIEM concepts.

---

## Tools Used

* Autopsy
* Sample Disk Image
* Windows Event Logs

---

## Topics Covered

### Incident Response Lifecycle

1. Preparation
2. Detection and Analysis
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

### Digital Forensics

* Evidence Collection
* Chain of Custody
* Disk Imaging
* Artifact Analysis

### Security Monitoring

* Log Analysis
* SIEM Systems
* Threat Detection

---

## Practical Activity

A sample disk image was loaded into Autopsy and analyzed. The investigation involved reviewing files, examining artifacts, and conducting keyword searches to identify potential evidence.
